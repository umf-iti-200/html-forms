Input

<input type="text">
Defines a single-line text input 
<input type="password">
Defines a password field 
<input type="email">
Used for input fields that should contain an email address. It depends on browser support 
<input type="tel">
Used for input fields that should contain a telephone number. It depends on browser support 
<input type="url">
Used for input fields that should contain a URL address. It depends on browser support 
<input type="radio">
Radio buttons let a user select ONLY ONE of a limited number of choices 
<input type="checkbox">
Checkboxes let a user select ZERO or MORE of a limited number of choices 
<input type="color">
Used for input fields that should contain a color. It depends on browser support 
<input type="date">
Used for input fields that should contain a date. It depends on browser support 
<input type="file">
Defines a file-select field and a "browser" button for file uploads

Select

<select>
    <option>Option 1</option>
    <option>Option 2</option>
    <option>Option 3</option>
</select>
Defines a drop-down list. For every option in the list, <option> is used

Textarea

<textarea rows="3" cols="15">
  
</textarea>
Defines a multi-line input field

Buttons

<button>Click me!</button>
Defines clickable button 
<button type="submit">Submit!</button>
Defines a submit button 
<button type="reset">Reset!</button>
Defines a reset button
